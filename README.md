# Resumo do projeto

## O problema 
Segundo o World Population Review os Estados Unidos possuem 653.1 mil pessoas que moram nas ruas. É importante investigar e visualizar como essa população se distribui pelo país,
como algumas regiões e estados têm um problema maior de população que mora nas ruas. 
Analisar se fatores familiares contribuem para uma maior propensão a ficar desabrigado, como pertencer ou não a uma família com filhos. Isso tudo contribui para que medidas públicas de apoio  sejam adaptadas para os seus contextos.

## Abordagem 
Foram analisados números brutos e a proporção de moradores de rua por Região/Estado assim como a categoria familiar do indivíduo influencia nos números.

## Resultados
*  A categoria familiar tem grande influência nos números de pessoas desabrigadas.
*  Algumas regiões tem uma população de pessoas que moram nas ruas muito maior que outras, então o problema de pessoas sem-teto não é igual no país como um todo.
*  A própria capital do país tem a pior proporção de moradores de rua por população total.
